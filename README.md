## Setup 

To setup this repo pls follow this steps - 

1. **Clone the Repository**: 
```bash 
git clone 
```
2. **Install Dependencies**: 

```bash 
cd groww_assignment  && npm install
```

3. **Start server**: 

```bash 
npm run dev
```
